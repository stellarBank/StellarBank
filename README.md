# StellarBank

Cross-border remittances powered by Stellar.

## Overview

StellarBank is a remittance platform built on the Stellar network that enables fast, affordable international money transfers for emerging markets. The platform leverages Stellar's blockchain infrastructure, stablecoins, and decentralized exchange capabilities to simplify cross-border payments while reducing costs and settlement times.

## Supported Corridors

* Nigeria ↔ United States
* Nigeria ↔ United Kingdom
* Africa ↔ Europe
* Additional global corridors planned

## Key Features

### Fast Transfers

Transfers settle within seconds using Stellar's blockchain infrastructure.

### Low Transaction Costs

Reduce remittance fees compared to traditional money transfer services.

### Multi-Currency Support

Support for stablecoins and local currency conversions.

### Mobile-First Experience

Designed for users in emerging markets with mobile accessibility as a priority.

### Cash-Out Network

Integration with local banking, mobile money, and payment partners.

### Security & Compliance

Built with security best practices and regulatory compliance considerations.

## Getting Started

### Requirements

* Node.js 16+
* Docker
* Docker Compose
* Stellar CLI

### Installation

```bash
git clone <repository-url>
cd StellarBank

docker-compose up -d

npm install
npm run dev
```
